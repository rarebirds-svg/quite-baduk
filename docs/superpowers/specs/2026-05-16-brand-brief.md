# Brand Brief — 새 브랜드 네이밍 (라운드 1)

> 사이트 리브랜딩을 위한 1차 정리. 이 문서는 **후보 발상 전 단계**의 제약·의미군·평가 기준만 담는다. 후보군은 라운드 2 산출물.

## 배경

현 사이트명 "K-Baduk"이 타사에서 이미 상표·서비스로 사용 중. 충돌 회피 + 우리 정체성에 맞는 새 워드마크 필요.

서비스 요약: KataGo Human-SL 기반의 1인용 바둑 사이트. 9급–9단 사다리, 18인 기풍 선택, 복기 모드의 승부처 분석 (학습 모드 + 패착/승착 분류).

## 확정 제약

| 항목 | 결정 |
|---|---|
| 톤 | 조용·문학적 |
| 시장 | 국내 우선 → 국제 |
| 도메인 | `.com` 우선 |
| 디자인 정체성 | Editorial Hardcover 연속 (종이·잉크·다도·서예·정원의 정적) |
| 시각 자산 | `font-serif` Newsreader, `font-sans` Pretendard, `font-mono` IBM Plex Mono. 색: paper/ink/oxblood/gold/moss |

## 미결 결정 (라운드 2 직전까지 정리 필요)

1. **언어 구성** — 한글 워드마크 + 영문 도메인 매칭 vs 영문 단어 단독 vs 신조어(합성).
2. **"Baduk/Go" 포함 여부** — 카테고리 명료성 ↔ 차별성.
3. **길이** — 영문 6자 이내 단음절 vs 2~3음절 합성.
4. **회사 ≠ 서비스** 분리 필요성 — 향후 컬렉션·학습 등 확장 가능성.
5. **발음 일관성 강도** — 한국인 발음과 로마자가 1:1 일치 강제 정도.
6. **부정 함의 회피** — 외국어 사전 스캔(영·일·중) 단계 포함.

## 선택된 의미군 (라운드 2에서 후보 생성 시 우선 탐색)

라운드 1 협의 결과, 다음 세 가족을 폭넓게 다룬다.

### a + c. 자연풍경 + 한국 문학·서예·다도
돌·물·우물·그늘·새벽·등불·안개·길·한지·먹·자작·매화·차밭·별·달
문방사우(붓·먹·종이·벼루)·묵향·정자(亭)·다실·서원
옛말: 한건(閑見)·청유(淸遊)·무현(無弦)·적묵(寂默)

### d + e. 고요·집중 + 추상·작은 사물
한(閑)·정(靜)·거(居)·휴(休)·침(沈)·관조·물러나기
한 수·두 점·빈 칸·행(行)·정(井)·국(局)

### f + g. 영문 추상 + 시각·매체 메타포
- monosyllable: kō · niri · hwa · won · mok · sok · gam
- Latin/Greek: lumen · vox · otium · kairos · ataraxia
- verbs: pause · dwell · settle · rest · draw
- compounds: –ink · –press · –journal · –leaf · –stone
- Editorial 연속성: Press · Journal · Folio · Quarterly · Chapbook · Manuscript

### 제외된 가족 (라운드 1 결정)

- **b. 바둑 술어/도구** (호선·정석·축·단수·화점 등) — 검색 친화적이지만 카테고리 명료성보다 차별성 약화가 더 큰 부담이라 제외.
- **h. KataGo/AI 직접 호명** — 시간성 약함, 5년 후 noise.

## 평가 기준

### 기본 6축

| 축 | 가중치 | 기준 |
|---|---:|---|
| 발음 | 30 | 한국인·외국인 모두 한 번에 발음 가능 |
| 기억성 | 20 | 1회 청취로 spelling 추측 가능 |
| 의미 적합 | 20 | 톤(조용·문학) + 도메인(바둑) 양쪽에 닿음 |
| 상표 위험 | 15 | KIPRIS · USPTO 동일·유사 부재 |
| 도메인 | 10 | `.com` 가용 또는 합리적 매물가 |
| 시간성 | 5 | 트렌드 어휘(–ify · –ai) 회피 |

### 보조 4축 (결정 갈림에 사용)

- **SEO/검색 가능성** — 일반어는 noise. 본 사이트는 트래픽 의도 적어 우선순위 낮음.
- **발음 일관성** — 한글/로마자 표기 1:1 일관.
- **부정 함의** — 외국어 사전 스캔.
- **확장성** — 후속 서비스 흡수할 우산성.
